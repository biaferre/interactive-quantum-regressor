# Interactive Quantum Regressor
This repository contains the code for the Interactive Quantum Regressor, an Open Quantum Systems-based regression model. It corresponds to the article "Interactive Quantum Regressor based on Open
Quantum Systems" written by Beatriz Férre, João V. M. Cardoso, Fábio Novaes and Fernando M. de Paula Neto and approved in BRACIS (Brazilian Conference for Intelligent Systems) 2026.
